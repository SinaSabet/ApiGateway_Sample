# API Gateway Sample

## 📋 Overview

A comprehensive implementation of a **.NET API Gateway** using **Ocelot** with **self-registration services**. This project demonstrates how to build a scalable, centralized entry point for microservices architecture using the Ocelot API Gateway framework.

## 🎯 Business Purpose

This project serves as:
- **API Gateway Pattern Implementation**: Centralized routing and request handling for microservices
- **Service Discovery Integration**: Automatic service registration without manual configuration
- **Educational Reference**: Best practices for implementing API Gateways in .NET
- **Enterprise Architecture**: Production-ready patterns for microservices communication

## 🏗️ Architecture

Simple ASCII diagram and explanation for routing, services, and self-registration.

## 🛠️ Technologies & Tools

- .NET (6/7/8)
- Ocelot
- Consul (optional) for service discovery
- JSON/YAML configuration

## 🚀 Features

- Centralized API routing
- Load balancing
- Request transformation
- Rate limiting
- Service discovery / self-registration

## 🚀 Quickstart

1. Clone
2. Configure ocelot.json and appsettings
3. dotnet restore && dotnet run
